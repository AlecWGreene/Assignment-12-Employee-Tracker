(async) viewInformationMenu()

Presents users with options to view Department, Employee, or Role tables. After selecting an option, the user can use "Select a different table" to reach this prompt again. After the user's selection, this method displays the data using tableRenderer, prompting the user to alter the view using the menu functions filterMenu, searchMenu, sortMenu, and columnsMenu. Function loops through alter view prompts until the user selects "Select a Different table", and then offers the view table prompt until it returns to mainMenu upon selection of the option "Return to Main Menu".
